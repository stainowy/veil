Welcome to Velang! This guide will help you install the compiler and write your first Velang program.

Installation

Prerequisites

  • Windows 10/11 (primary target)
  • Rust toolchain (for building from source)
  • Visual Studio Build Tools or equivalent C compiler

Install from Source

$ git clone https://github.com/yourusername/velang.git $ cd velang $ cargo build --release
Note: The Velang compiler is built in Rust and generates C code that is then compiled to native executables.

Your First Program

Create a file called hello.ve:

fn main() { println("Hello, World!"); }

Compile and run it:

$ ve run hello.ve Hello, World!

Basic Syntax

Variables

fn main() { let name = "Velang"; let version = 1; let is_fast = true; println("Language: " + name); println("Version: " + version); }

Functions

fn add(a: i32, b: i32) -> i32 { return a + b; } fn main() { let result = add(5, 3); println("5 + 3 = " + result); }

Structs

struct Person { name: string, age: i32 } fn main() { let person = Person { name: "Alice", age: 30 }; println("Name: " + person.name); println("Age: " + person.age); }
Alpha Version: Velang is in early development. Some features may be incomplete or subject to change.
